# RC505mk2 Effect Memo

This is a simple web application to take notes of effect settings for the BOSS RC-505mk2 Loop Station.

## Features

- Create and save effect presets for both INPUT FX and TRACK FX.
- Select effects from a predefined list that matches the RC-505mk2.
- Automatically displays the parameters for the selected effect.
- Sequencer parameters are automatically added for effects that support them.
- Save presets with a custom name.
- Search saved presets by name.

## How to Use

1. Open the `index.html` file in your web browser.
2. Select whether you want to create a preset for "INPUT FX" or "TRACK FX".
3. For each of the four effect slots (A, B, C, D), select an effect from the dropdown menu.
4. Fill in the values for the parameters that appear for each selected effect.
5. Enter a name for your preset.
6. Click the "Save Preset" button.
7. Your saved presets will appear in the "Saved Presets" list.
8. You can use the search bar to filter your saved presets by name.
